=============================

Android SDK Download
=====================
Link : https://developer.android.com/studio
Pilih yang : Khusus alat command-line, : 	commandlinetools-win-14742923_latest.zip
Buat Folder "Android" di C:\
Ekstrak commandlinetools-win-14742923_latest.zip ke foler C:\Android
Buat folder "latest" di "C:\Android\cmdline-tools"
Masukan semua file disana kedalam folder "latest", C:\Android\cmdline-tools\latest
Buat folder "platform-tools" di "C:\Android\platform-tools"

Environment Variable > PATH > Edit
Masukan : C:\Android\cmdline-tools\latest\bin

New :
ANDROID_HOME
C:\Android

ANDROID_SDK_ROOT
C:\Android

> sdkmanager --sdk_root=C:\Android "platform-tools" "platforms;android-36" "build-tools;36.0.0"
